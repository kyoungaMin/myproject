# 🌙 당신이 잠든 사이 (While You Were Sleeping)
## 미국주식 데일리 브리핑 서비스 기획서

---

## 1. 서비스 개요

**"당신이 잠든 사이"**는 한국인 투자자들이 잠든 밤 시간 동안 열리는 미국 주식시장의 핵심 동향을 매일 아침 자동으로 전달하는 AI 기반 브리핑 서비스입니다. Yahoo Finance 데이터를 기반으로 거래량과 상승률이 높은 화제 종목을 자동 선정하고, 관련 뉴스와 정보를 수집하여 AI가 시각적으로 정리된 브리핑 이미지를 생성합니다. 사용자는 별도의 입력이나 조작 없이 매일 아침 이메일 또는 슬랙을 통해 전날 밤 미국 시장의 핵심 정보를 한눈에 확인할 수 있습니다. 바쁜 한국 직장인들이 출근길에 1분 만에 미국 주식 시장 동향을 파악할 수 있도록 설계되었습니다.

---

## 2. 핵심 가치 제안 (Value Proposition)

| 구분 | 내용 |
|------|------|
| **For** | 미국 주식에 투자하지만 시장 분석에 시간을 쏟기 어려운 한국인 직장인 |
| **Who** | 매일 아침 미국 시장 동향을 빠르게 파악하고 싶은 투자자 |
| **Our Service** | 복잡한 분석 과정 없이 "가장 활발한 종목"을 자동 선정하여 AI 브리핑 제공 |
| **That** | 시차로 인한 정보 격차를 해소하고, 투자 의사결정에 필요한 핵심 정보만 전달 |
| **Unlike** | 직접 검색하거나 복잡한 차트를 분석해야 하는 기존 투자 정보 서비스 |
| **We** | 100% Chatless 자동화로 사용자 개입 없이 매일 아침 브리핑 전달 |

### 💡 핵심 차별점

- 🎯 **Chatless Experience** - 입력 없이 자동 발송
- 📊 **데이터 기반 종목 선정** - 주관적 판단 배제
- 🖼️ **AI 이미지 브리핑** - 1분 내 핵심 파악 가능
- ⏰ **시차 극복** - 한국 시간 기준 최적의 발송 타이밍

## 3. 주요 기능 목록

### 3.1 🔥 화제 종목 자동 선정
- Yahoo Finance 실시간 데이터 수집
- 거래량 급증 종목 TOP 5 선정
- 상승률 상위 종목 TOP 5 선정
- 중복 제거 및 최종 브리핑 종목 확정 (일 5~10개)

### 3.2 📰 종목별 정보 수집
- 기업 기본 정보 (시가총액, 섹터, 52주 최고/최저)
- 당일 주가 변동 정보 (시가, 종가, 변동률, 거래량)
- 관련 최신 뉴스 헤드라인 수집 (영문 → 한글 번역)

### 3.3 🎨 AI 이미지 브리핑 생성
- 시각적으로 정리된 인포그래픽 스타일 이미지 자동 생성
- 종목별 핵심 정보 한눈에 파악 가능한 레이아웃
- 브랜드 아이덴티티 반영 일관된 디자인

### 3.4 📧 자동 발송 시스템
- 이메일 발송 (개인 사용자용)
- 슬랙 웹훅 발송 (팀/그룹용)
- 한국 시간 오전 7-8시 자동 발송

### 3.5 📅 아카이브 & 히스토리
- 과거 브리핑 기록 저장
- 특정 종목 언급 히스토리 조회
- 주간/월간 트렌드 리포트 (추후 확장)

### 3.6 ⚙️ 구독 관리
- 이메일/슬랙 구독 신청 및 해지
- 발송 시간 커스터마이징 (선택)
- 관심 섹터 필터링 설정 (선택)

### 3.7 🔔 특별 알림 (확장 기능)
- 급등/급락 종목 실시간 알림
- 보유 종목 등록 시 맞춤 알림
- 실적 발표 일정 사전 알림

## 4. 사용자 시나리오 (User Journey)

### 페르소나: 김직장 (32세, IT 회사 개발자)
> "미국 주식에 관심은 있는데, 밤에 시장 보면서 잠을 설치기 싫어요. 출근길에 핵심만 빠르게 알고 싶어요."

### User Journey Map

```
[STEP 1] 서비스 발견 & 구독
    │
    │  • 동료 추천으로 서비스 알게 됨
    │  • 랜딩 페이지에서 이메일 입력
    │  • 구독 확인 메일 수신 → 구독 완료
    │
    ▼
[STEP 2] 첫 브리핑 수신 (D+1 아침 7시)
    │
    │  📧 "오늘의 미국 시장 브리핑이 도착했습니다"
    │  • 이메일 오픈
    │  • AI 생성 이미지 브리핑 확인
    │  • 화제 종목 5개 + 핵심 뉴스 요약 확인
    │
    ▼
[STEP 3] 일상 속 활용
    │
    │  ☀️ 출근길 지하철에서 (약 1분 소요)
    │  • 어제 밤 NVIDIA 15% 급등 확인
    │  • "AI 반도체 수요 증가" 뉴스 헤드라인 확인
    │  • 관심 종목으로 추가할지 메모
    │
    ▼
[STEP 4] 투자 의사결정 참고
    │
    │  💼 점심시간 또는 퇴근 후
    │  • 아침에 본 종목 추가 리서치
    │  • 투자 결정 시 참고 자료로 활용
    │  • 브리핑에서 본 종목 매수/관심 등록
    │
    ▼
[STEP 5] 지속적 사용 & 공유
    │
    │  📢 만족도 높을 경우
    │  • 팀 슬랙 채널에 서비스 공유
    │  • 동료들과 아침 브리핑 내용으로 대화
    │  • 투자 동호회에 추천
```

### 핵심 터치포인트별 경험

| 단계 | 사용자 행동 | 감정 | 서비스 제공 가치 |
|------|------------|------|-----------------|
| 구독 | 이메일 입력 (10초) | 기대감 | 간편한 온보딩 |
| 수신 | 이메일 확인 | 호기심 | 적절한 타이밍 |
| 열람 | 이미지 브리핑 확인 (1분) | 만족감 | 핵심 정보 요약 |
| 활용 | 투자 의사결정 참고 | 신뢰감 | 데이터 기반 정보 |
| 공유 | 동료에게 추천 | 자부심 | 공유할 만한 가치 |

## 5. 성공 지표 (KPI)

### 📊 핵심 KPI 대시보드

| # | KPI | 정의 | 목표 (6개월) | 측정 방법 |
|---|-----|------|-------------|----------|
| 1 | **구독자 수** | 총 활성 구독자 수 | 10,000명 | 이메일 + 슬랙 채널 합산 |
| 2 | **이메일 오픈율** | 발송 대비 열람 비율 | 45% 이상 | 이메일 트래킹 (픽셀/링크) |
| 3 | **DAU/MAU 비율** | 월간 사용자 중 일일 활성 비율 | 60% 이상 | 일일 오픈 유저 / 월간 유저 |
| 4 | **구독 유지율** | 30일 후 구독 유지 비율 | 80% 이상 | (구독자 - 해지자) / 구독자 |
| 5 | **NPS (순추천지수)** | 서비스 추천 의향 점수 | 50점 이상 | 월 1회 설문조사 |

### 📈 보조 지표

| 지표 | 설명 | 목표 |
|------|------|------|
| 평균 열람 시간 | 브리핑 콘텐츠 소비 시간 | 1분 30초 이상 |
| 공유율 | 브리핑 공유 비율 | 5% 이상 |
| 슬랙 리액션 수 | 슬랙 발송 시 이모지 반응 | 메시지당 평균 3개 이상 |
| 오류 발생률 | 발송 실패 또는 데이터 오류 | 0.1% 미만 |
| 피드백 응답률 | 사용자 피드백 수집 비율 | 월 구독자의 10% |

### 🎯 단계별 목표

**[Phase 1: MVP] 0-3개월**
- 구독자 1,000명 확보
- 이메일 오픈율 40% 달성
- 일일 자동 발송 안정화 (99.9% 가용성)

**[Phase 2: Growth] 3-6개월**
- 구독자 10,000명 확보
- 슬랙 발송 기능 출시
- NPS 50점 달성
- 주간 리포트 기능 추가

**[Phase 3: Scale] 6-12개월**
- 구독자 50,000명 확보
- 프리미엄 구독 모델 도입
- 모바일 앱 출시 검토
- B2B 기업 구독 상품 출시
